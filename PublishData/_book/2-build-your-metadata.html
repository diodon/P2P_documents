2 Build your metadata

As it was mentioned, this step is as important as the colection of the data itself. You will fill the metadata section of the IPT instalation of the MBON project.

  1. Login into the MBON IPT instalation using the credentials we have set for you

NOTE: Your user name is your email address

  1. You need to create a new resource (this is how IPT calls the data sets). Go to the Manage Resources tab and at the very bottom fill the fields of the Create a new Resource section.
  • Name your resource. It has to be a short name with no spaces or special characters. Donâ€™t worry too much as we will change the name to a proper one later.
  • Select the type of the resource as EVENT data. This is very important as your data comply with this schema.
  • Click on Create new Resource. You will be redirected to the ressource page you just created.
During the creation/edition process the data set (the resource) will be only visible to you until you finaly make it public.
